Lesson 3 - Getting the LED to change color and blink a set number of times then change colors.

Objective

In this lesson you will be learning how to control the color and the number of times the LED will blink. This is simulates what a cross walk light might be like. Get the LED to be green for 3 seconds, and then blink red for three seconds.


                

                    #!/usr/bin/env python3


                    # Filename: Lesson03.py

                    import linkbot

                    import time



                    robot = linkbot.Linkbot('')
                        #set ID to a certain Linkbot


                

                robot.setLedColor({{m.led1[0]}},
                {{m.led1[1]}},
                {{m.led1[2]}})
                   #sets color of LED to green

                time.sleep({{m.sleep1}})
                                  #waits 3 seconds

                for i in range(1,
                {{m.cnt}}):
                          #runs the block that follows five times

                    robot.setLedColor(255,0,0)
                 #sets color to red

                    time.sleep({{m.sleep2}})
                            #waits half a second

                    robot.setLedColor(0,0,0)
                   #turns off LED

                    time.sleep({{m.sleep3}})
                            #waits half a second

                    robot.setLedColor(0,0,255)
                 #turns LED to blue

                    time.sleep({{m.sleep4}})
                            #waits half a second
